# Проект: "God of War"

Цель проекта : "God of War" реализован для улучшения навыков вёрстки и наполнения портфолио.

## Технологии, которые использовались

### HTML

1. SEO
2. Open Graph теги
3. Семантическая верстка
4. Pixel Perfect верстка

### CSS

1. Методология БЭМ
2. Флексбокс верстка
3. Адаптивная верстка
4. Грид верстка
5. Медиазапросы
